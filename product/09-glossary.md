# Glossary

- Vendor: a service provider for an event category (DJ, photographer, etc.).
- Event host: consumer planning an event.
- Event planner: professional coordinator managing vendors across events.
- Quote request: structured request sent to one or more vendors for an event.
- Offer: a vendor’s response to a quote request (price + scope + terms + availability).
- Booking: confirmed commitment between user and vendor to provide service at a specific event.
- Contractor match: (if used) the chosen vendor/offer that best fits an event request.
